I've gathered some practical tips while building documentation sites with Starlight.

## Custom Styles

Import custom CSS files via `customCss` in `astro.config.mjs` to override default styles.

## Component Usage

Starlight provides rich built-in components such as Card, Steps, LinkCard, etc., which can be used directly in MDX.

## Multilingual Support

Starlight supports i18n configuration, making it easy to build multilingual documentation sites.
