## What is Astro

Astro is a modern static site generator with a content-first, zero-JS-by-default philosophy.

## Core features

- **Islands architecture** - Load interactive components on demand
- **Multi-framework support** - Integrate React, Vue, Svelte, and more
- **Markdown first** - Native MDX support

## With Starlight

Starlight is a documentation site framework built on Astro, ready to use out of the box.
